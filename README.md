# azcam-vattspec

## Purpose

This repository contains the *azcam-vatt* *azcam* environment. It contains code and data files for the VATT VattSpec camera system maintained by the UArizona Imaging Technology Laboratory.

## Installation Example

Azcam code is typically run directly from the installation folders to allow changes. From `c:\azam` folder...

```
git clone https://github.com/uaitl/azcam-vattspec
pip install -r azcam-vattspec\requirements.txt
```

For Windows systems the normal destination is `c:\azcam\azcam-vattspec`.
